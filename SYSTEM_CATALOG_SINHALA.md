## පද්ධති දළ විශ්ලේෂණය
**Nexus Learning Management System (LMS)** යනු ප්‍රධාන පරිශීලක භූමිකාවන් තුනකට සහය දක්වන වෙබ් පාදක වේදිකාවකි:
- **සිසුන් (Students)** – පාඨමාලා නැරඹීම, ප්‍රගතිය පරීක්ෂා කිරීම, ප්‍රමුඛ පුවරු (Leaderboards) බැලීම සහ දැනුම්දීම් ලබා ගැනීම.
- **ගුරුවරුන් (Teachers)** – පාඨමාලා නිර්මාණය, පැවරුම් (Assignments), ප්‍රශ්නාවලි (Quizzes) කළමනාකරණය, ලකුණු ලබා දීම සහ වාර්තා සකස් කිරීම.
- **පරිපාලකයින් (Admins)** – පරිශීලකයින්, ගෙවීම්, ලියාපදිංචි කිරීම් සහ පද්ධති සැකසුම් කළමනාකරණය.

මෙම පද්ධතිය HTML, CSS සහ JavaScript භාවිතයෙන් ඉදිරිපස (Frontend) ලෙසද, **Supabase** පසුතල (Backend) ලෙසද නිමවා ඇත.

---

## ගෘහ නිර්මාණ ශිල්පය (Architecture)
```
+---------------------------------------------------+
|                     පරිශීලකයා                     |
|        (HTML, CSS, JS – Vite හරහා ක්‍රියාත්මක වේ)    |
+----------------------+----------------------------+
                       | HTTP ඉල්ලීම් (Requests)
+----------------------+----------------------------+
|                 Supabase Backend                 |
|  • PostgreSQL දත්ත සමුදාය                           |
|  • Auth (පරිශීලක සත්‍යාපනය)                        |
+----------------------+----------------------------+
```

## මුහුණත සංරචක (Frontend Components)
- **HTML Templates** – පද්ධතියේ විවිධ පිටු සඳහා භාවිතා වේ.
- **CSS Design** – `css/dashboard.css` මගින් නවීන, අලංකාර (Glassmorphism) පෙනුමක් ලබා දෙයි.
- **JavaScript** – දත්ත ලබා ගැනීම (Data fetching) සහ අන්තර්ක්‍රියාකාරී බව (Interactivity) ලබා දීම සඳහා භාවිතා කරයි.

---

## පසුතල ඒකාබද්ධතාවය (Supabase Integration)
- **Auth** – පරිශීලක ලොග් වීම සහ ආරක්ෂාව Supabase හරහා සිදුවේ.
- **Database** – PostgreSQL භාවිතයෙන් දත්ත ගබඩා කිරීම සිදු කරයි.
- **Real-time** – ප්‍රමුඛ පුවරු සහ දැනුම්දීම් සඳහා තථ්‍ය කාලීන යාවත්කාලීන කිරීම්.

---

## දත්ත සමුදාය (Database Schema)
ප්‍රධාන වගු (Tables):
- `users`: පරිශීලක තොරතුරු.
- `courses`: පාඨමාලා විස්තර.
- `enrollments`: ලියාපදිංචි වීමේ දත්ත.
- `submissions`: සිසුන් ඉදිරිපත් කරන පැවරුම්.

---

## අනාගත දියුණු කිරීම්
- **ජංගම යෙදුම (Mobile App)** – පද්ධතිය ජංගම දුරකථන සඳහා වඩාත් සුදුසු ලෙස සකස් කිරීම.
- **විශ්ලේෂණ (Analytics)** – සිසුන්ගේ අධ්‍යාපන ප්‍රගතිය පිළිබඳ සවිස්තරාත්මක දත්ත ලබා ගැනීම.
- **සජීවී කතාබහ (Live Chat)** – ගුරුවරුන් සහ සිසුන් අතර සන්නිවේදනය දියුණු කිරීම.
